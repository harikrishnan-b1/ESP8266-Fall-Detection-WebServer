# ESP8266 Fall & Motion Detection System

This project uses the **DFRobot C1001 Human Detection Radar Sensor** with **ESP8266** to detect:
- Human presence
- Motion level
- Fall detection

A **web dashboard** displays live sensor data over Wi-Fi.

## Hardware Used
- ESP8266 (NodeMCU)
- DFRobot C1001 Radar Sensor
- Wi-Fi Network

## Features
- Real-time fall detection
- Auto-refresh web interface
- Motion intensity monitoring
- Presence detection

## Libraries Required
- DFRobot_HumanDetection
- ESP8266WiFi
- ESP8266WebServer
- SoftwareSerial

## How to Use
1. Upload code to ESP8266
2. Connect to Wi-Fi
3. Open browser with ESP8266 IP address
4. View live data
